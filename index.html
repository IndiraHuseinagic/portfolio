Xshoes - Online Shop

Technologies and tools that I use:

Xshoes Web App

Xshoes is a web application designed for online shopping. It is dynamic web app where user can interact and perform operations such as search, login, place order, make favorite list, make shopping cart list, filter, sort, and so on. Website is responsive providing an optimal viewing experience across a wide range of devices such as smartphones, tablets, desktop computer monitors, etc. App uses Firebase as backend service for hosting, cloud storage, real-time database, user authentication and website deployment.

✓ Client-Side: HTML 5, CSS 3, Bootstrap 4, TypeScript 4.1.2
✓ Framework: Angular 11.2.2
✓ Back-end Side: Firebase
