Xshoes Web App

Xshoes is a web application designed for online shopping. It is dynamic web app where user can interact and perform operations such as search, login, place order, make favorite list, make shopping cart list, filter, sort, and so on. Website is responsive providing an optimal viewing experience across a wide range of devices. App uses Firebase as backend service for hosting, cloud storage, real-time database, user authentication and website deployment.

Modern Home Web App

Modern Home is a web application for online apartment reservation.It is dynamic web app where user can interact and perform operations such as filter, search, register, login, logout, make reservation, cancel reservation. Website is responsive providing an optimal viewing experience across a wide range of devices. App uses Node with Express for building RESTful services, and MongoDB database for storing data. App is deployed on Heroku platform.

Modern Home ist eine Webanwendung für Online-Wohnung Reservierung. Es ist eine dynamische Web-App, in der der Benutzer interagieren und Vorgänge ausführen kann, wie z. B.: suchen, filtern, registrieren, anmelden, abmelden, Reservierung machen, Reservierung stornieren. Die Website ist responsive und bietet ein optimales Seherlebnis auf einer Vielzahl von Geräten. Die App verwendet Node mit Express zum Erstellen von RESTful-Services und die MongoDB-Datenbank zum Speichern von Daten. Die App wird auf der Heroku-Plattform bereitgestellt.
