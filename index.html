Xshoes Web App

Xshoes is a web application designed for online shopping. It is dynamic web app where user can interact and perform operations such as search, login, place order, make favorite list, make shopping cart list, filter, sort, and so on. Website is responsive providing an optimal viewing experience across a wide range of devices. App uses Firebase as backend service for hosting, cloud storage, real-time database, user authentication and website deployment.

Xshoes ist eine Web app für Online-Shopping. Es handelt sich um eine dynamische Web-App, in der der Benutzer interagieren und Vorgänge ausführen kann, wie z. B.: Suche, Anmeldung, Bestellung aufgeben, Wunschliste erstellen, Warenkorbliste erstellen, filtern, sortieren und so weiter. Webseite ist responsive und bietet ein optimales Seherlebnis auf einer Vielzahl von Geräten. App verwendet Firebase als Backend für Hosting, Cloud-Speicher, Echtzeit-Datenbank, Benutzerauthentifizierung und Website-Bereitstellung.

✓ Front-end: HTML 5, CSS 3, Bootstrap 4, TypeScript 4.1.2
✓ Framework: Angular 11.2.2
✓ Back-end: Firebase 9.3.0
